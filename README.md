# Projeto Final — Desenvolvimento de Jogos

Este projeto tem como objetivo o desenvolvimento de um Produto Mínimo Viável (MVP) de um jogo digital, contemplando mecânica, narrativa e estética.

O foco principal está na construção de uma experiência clara e funcional, garantindo que o usuário consiga compreender e interagir com o jogo de forma simples e eficiente.

O projeto também envolve a definição da Arquitetura da Informação (AI), considerando a organização do conteúdo e a navegação dentro do sistema.
